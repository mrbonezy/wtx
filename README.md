# wtx

`wtx` is a small CLI for managing and jumping between git worktrees with a fast, interactive picker.

## Install

```sh
curl -fsSL https://raw.githubusercontent.com/mrbonezy/wtx/main/scripts/install.sh | sh
```

## Usage

```sh
wtx
wtx setup
wtx version
wtx check
```

## Build

```sh
go build -o wtx
```
